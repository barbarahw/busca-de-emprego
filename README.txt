🧭 Busca de Emprego — API REST

API desenvolvida em Spring Boot para gerenciar usuários e autenticação em um sistema de busca de empregos. O sistema permite o cadastro, login, leitura, edição e exclusão de perfis, além de logout com autenticação via token.

⚙️ Como Executar o Projeto

Pré-requisitos: Java 17 ou superior instalado Maven instalado
Passos:

1. Clonar o repositório
git clone https://github.com/barbarahw/busca-de-emprego.git

2. Acessar a pasta do projeto
cd busca-de-emprego

3. Compilar e rodar com Maven
mvn spring-boot:run
